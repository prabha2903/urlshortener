# URL Shortener App 🚀
A simple and efficient URL shortener web application that converts long URLs into short, shareable links. Built using React + Node.js + Express.js + MongoDB.

##  Architecture
Frontend (React) → Backend (Node.js + Express) → Database (MongoDB) → Redirect Service

##  Features
Convert long URLs into short links  
- ⚡ Instant short URL generation  
- ✏️ Custom alias support  
- ⏳ Link expiry option  
- 📁 Manage all links in table view  
- 📋 Copy, edit, delete, and QR code generation  
- 📈 Analytics: clicks, browser & device tracking, visit history  
- 🚀 Deployed full-stack application
